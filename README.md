# MyStore - 商品展示网页

一个简洁且富有创意的商品展示网页，展示了现代化的Web设计。

## 📋 项目介绍

MyStore 是一个专业的商品展示平台网页，包含以下主要功能：

- **导航栏** - 快速导航到不同页面区域
- **英雄部分** - 吸引人的首页介绍
- **商品展示** - 精选商品卡片展示
- **特色介绍** - 平台优势展示
- **联系表单** - 客户联系方式
- **响应式设计** - 完美适配各种设备

## 🎨 设计特点

- ✨ 现代化渐变色设计
- 📱 完全响应式布局
- 🎯 简洁清晰的信息架构
- 💫 平滑的过渡动画效果
- 🎭 创意的组件设计

## 🚀 快速开始

### 本地预览

1. 克隆或下载此项目
2. 用浏览器打开 `index.html` 文件即可查看网页效果

### 在线部署

#### 方法一：使用 GitHub Pages

1. 进入仓库的 Settings（设置）
2. 向下滚动到 "Pages" 部分
3. 在 "Source" 中选择 `main` 分支
4. 点击 Save
5. 访问 `https://Cabbage616.github.io/my-store` 查看网页

#### 方法二：其他托管服务

可以将文件上传到以下服务：
- Netlify
- Vercel
- Firebase Hosting
- 阿里云 OSS
- 腾讯云 COS

## 📄 页面内容

### 1. 导航栏

## 🔧 自定义指南

### 修改颜色

编辑 `styles.css` 中的 CSS 变量部分：

```css
:root {
    --primary-color: #6366f1;      /* 主色 */
    --secondary-color: #ec4899;    /* 辅助色 */
    --dark-color: #1f2937;         /* 深色 */
    --light-color: #f9fafb;        /* 浅色 */
}
```

### 更改商品信息

编辑 `index.html` 中的商品部分，修改以下内容：
- 商品图片 URL
- 商品名称
- 商品价格
- 商品描述
- 功能列表

### 修改公司信息

在 `index.html` 的联系部分修改：
- 邮箱地址
- 电话号码
- 公司地址
- 社交媒体链接
